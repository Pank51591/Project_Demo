Convert the result of training by the neural network into C source code, providing a C source file that contains a program for handling gesture input.
Please make sure to output the file when the gesture configuration is changed.


File output :
You can specify any location for the file output, but by default it is output to the following location.

Output location :

  • <project>\qe_gen

Output files :

  • Translator folder (Files output by e-AI translator)
  • qe_gesture_user.c
  • qe_gesture.c
  • qe_gesture.h


Notes on C source generation:

  • If there is no license file for the e-AI translator, or if the license file is out of date, the e-AI translator will fail to run. Please refer to the guide "Workflow: Install Environment" to obtain the license file.

  • If e-AI translator fails to run, a log file will be output to the following location
    <e2 studio install path>\eclipse\translator_xx_xx_xxxx.log
