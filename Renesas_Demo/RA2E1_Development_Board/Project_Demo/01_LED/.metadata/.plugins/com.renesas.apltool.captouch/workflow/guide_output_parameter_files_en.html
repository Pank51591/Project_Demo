The result of tuning is output as a parameter file and can be read by touch middleware.
Please make sure to output the file when tuning is performed and when the touch interface configuration is changed.


File output :
You can specify any location for the file output, but by default it is output to the following location.

Output location :

  • <project>\qe_gen

Output files :

  • qe_touch_define.h
  • qe_touch_config.h
  • qe_touch_config.c
