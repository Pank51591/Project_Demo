ジェスチャ開発に使用する環境一式をお使いのPCにダウンロードします。
[ジェスチャ環境を導入する]ボタンを押して表示される[ジェスチャ環境の導入状況]ダイアログにて、[ダウンロード]ボタンを押してください。
環境のダウンロードは、初めてジェスチャ開発を行う場合、環境の更新がある場合のみ実行してください。


導入時の注意点：

  • 環境をダウンロードするには、お使いのPCに数GBの空き容量が必要です。
  • 環境にはルネサス製 AI変換ツールであるe-AIトランスレータ(コマンドライン版)が含まれています。e-AIトランスレータを利用するにはライセンスファイルが必要となりますので、Webページから入手後、指定のフォルダに置いてください。ライセンスファイルは一定期間ごとに更新が必要です。

    Webページ:
    https://www.renesas.com/application/technologies/e-ai/tool

    フォルダ:
    <サポートフォルダのダウンロードエリア>\QE\AI_Package[index]\Tools\eAITranslator[バージョン]\bin

    サポートフォルダのダウンロードエリアの場所は以下の手順で確認できます。
    1. e2 studioの「ヘルプ(H)」 - 「e2 studio について(A)」メニューをクリック。
    2. 「e2 studio について」ウィンドウの「インストール詳細(i)」ボタンをクリック。
    3. 「e2 studio のインストール詳細」ウィンドウの「Support Folders」タブを表示し、「e2 studio download area」のパスを確認。
