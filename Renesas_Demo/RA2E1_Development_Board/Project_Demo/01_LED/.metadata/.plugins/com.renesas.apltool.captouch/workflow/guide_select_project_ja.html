静電容量タッチを組み込むe2 studio用のプロジェクトを選択します。
プロジェクトはe2 studioのプロジェクト作成機能にて作成します。この時、スマート・コンフィグレータを利用するプロジェクトにします。
プロジェクト作成後に、スマート・コンフィグレータで、以下の設定を行います。

  • 周辺モジュールクロック(PCLKB、PCLKLまたはFCLK)の設定
  • 静電容量タッチドライバ/ミドルウェアの追加
  • 使用するTS端子の有効設定


調整の条件：
タッチセンサの調整に必要なプロジェクトの条件は、以下の通りです。

  • タッチセンサ端子の設定が完了していること。
  • スマート・コンフィグレータにて作成していること。
  • ビルドが通り、main関数の先頭まで実行できること。
  • 周辺モジュールクロック(PCLKB、PCLKLまたはFCLK)の設定がプログラム中で完了していること。
