学習済みのAIモデルをCソースに変換し、ジェスチャ用プログラムとともにファイルを生成します。
ジェスチャ構成を変更した場合は、必ずファイルを出力してください。


ファイル出力：
ファイルの出力先は任意の場所を指定できますが、デフォルトでは以下に出力されます。

出力先：

  • <プロジェクト>\qe_gen

出力ファイル / フォルダ：

  • Translatorフォルダ (e-AIトランスレータが出力したファイル群)
  • qe_gesture_user.c
  • qe_gesture.c
  • qe_gesture.h


Cソース生成時の注意：

  • e-AIトランスレータのライセンスファイルがない場合、ライセンスファイルが古い場合はe-AIトランスレータの実行に失敗します。[ワークフロー: ジェスチャ環境の導入]のガイドを参考に、ライセンスファイルを入手してください。

  • e-AIトランスレータの実行に失敗した場合は、以下の場所にログファイルが出力されます。
    <e2 studioのインストール先>\eclipse\translator_xx_xx_xxxx.log
