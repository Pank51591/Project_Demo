Perform tuning of each touch sensor automatically using serial (UART) connection.
To perform tuning using serial connection, please follow the steps below.

  1. Configure settings for serial connection in the Smart Configurator. (Add drivers, configure pins, etc...)
  2. Configure the touch middleware for serial connection.
  3. Click on the "Output Files" button to output the files required for a tuning process.
  4. Implement a process to call the main loop for touch in the main() function.
  5. Build the project and run the program.
  6. Specify the baud rate and port number, and click on the "Start Tuning" button to start tuning.


Tuning procedure :

  1. Click on the "Start Tuning" button. The "Automatic Tuning Processing" Dialog Box will be displayed.
  2. Follow the instructions displayed in the "Automatic Tuning Processing" Dialog Box to start tuning.
  3. For example, in the tuning of touch-interface buttons, automatic tuning of a single button involves one time measurement. Touch the target button according to the instructions shown in this dialog. After that, press any key (e.g. Y) on the keyboard.
  4. After the tuning of all touch interfaces is finished, the result will be displayed. Close the "Automatic Tuning Processing" Dialog Box if there are no problems. Otherwise (e.g. when you have failed to touch the touch interface in the tuning process), select the target touch interface and press the "Retry" button to start re-tuning.


Notes on tuning:

  • If "Advanced mode" is checked, settings related to tuning can be made. However, if you will set these values inadvertently or without clear understanding, it could lead to poor tuning results.
