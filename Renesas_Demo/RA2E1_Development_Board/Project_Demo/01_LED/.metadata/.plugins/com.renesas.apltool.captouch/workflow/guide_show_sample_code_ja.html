静電容量タッチを使用するためのコードを実装します。
[例を表示する]ボタンを押すとサンプルコードが表示されますので、お使いのプロジェクトのmain関数に実装してください。
サンプルコードはタッチインタフェースの構成によって変わりますので、タッチインタフェース構成を変更した場合は、必ず実装も変更してください。


ファイル出力：
サンプルコードはファイルとしても出力可能です。その場合は、main関数内でqe_touch_main関数を呼び出してください。

出力先：

  • <プロジェクト>\qe_gen

出力ファイル：

  • qe_touch_sample.c


ジェスチャ使用時の注意：
ジェスチャを使用する場合は、以下を行ってください。

  • 数学ライブラリ(math.h)を使用する設定にしてください。
  • ヒープサイズおよびスタックサイズは必要に応じてサイズを大きくしてください。
  • RAマイコン使用時にはAI前処理でランダム関数を使用するため、以下のチェックを外すかヒープサイズを大きくしてください。
    - GNU ARM Cross C Linkerオプション － [Use newlib-nano(--specs=nano.specs)] チェックを外す。
