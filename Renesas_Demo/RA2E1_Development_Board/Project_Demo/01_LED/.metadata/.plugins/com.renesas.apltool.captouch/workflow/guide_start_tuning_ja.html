エミュレータ接続にて各タッチセンサの調整を自動で行います。
[調整を開始する]ボタンを押し、表示される手順に従って操作を行ってください。


調整の流れ：

  1. [調整を開始する]ボタンを押します。しばらくすると調整処理に入り、[自動調整処理中]ダイアログが表示されます。
  2. [自動調整処理中]ダイアログの指示に従い調整を行います。
  3. 例えばボタンの自動調整では、1つのボタンに対し1回の測定を行います。 ダイアログの指示に従い、対象のボタンに触れて、PCのキーボードのいずれかのキー(例えばYキー)を押してください。
  4. 3Dジェスチャ(AI)の自動調整では、3Dジェスチャを認識させたい高さに手を置き、PCのキーボードのいずれかのキーを押してください。
  5. すべてのタッチインタフェースの測定が終ったら結果が表示されます。問題がなければ完了してください。 タッチインタフェースに触らずにPCのキーを押してしまった場合などは、再測定をするタッチインタフェースを選択後に[リトライ]ボタンを押し、再測定をしてください。


調整時の注意点：

  • [調整を開始する]ボタンを押す前には、 リセットベクタからmain関数まで実行できるようになっていること、 エミュレータからターゲットボードが切断されていることを確認してください。
  • main関数までの実行は、必ずしもmain関数である必要はありません。 ユーザアプリケーションの入り口で実行が停止するように、 e2 studioのデバッグ構成で設定をしてください。
  • [アドバンスドモード(高度な設定)]にチェックを入れると、調整に関する設定を行うことができます。ただし、誤った値を設定した場合や、明確な理解なしに設定した場合は、調整結果が悪くなる可能性があります。
