タッチインタフェース構成の選択を行います。
タッチインタフェース構成を未作成の場合は、[タッチインタフェース構成の新規作成]をリストから選択します。表示される[タッチインタフェース構成の作成]ダイアログでタッチインタフェースを配置し、TS端子を割り当てます。


タッチインタフェース構成作成のヒント：

  • タッチインタフェースの配置は、繰り返し配置領域をクリックすることで複数配置ができます。
  • タッチインタフェースの配置を終了する場合は、タッチインタフェース名のボタン(ダイアログ右側)をもう1度押すか、ESCキーを押します。
  • SHIFTキーを押しながら配置領域をクリックすることで、配置を終了することも可能です。
  • 配置したタッチインタフェースをダブルクリックし、 [タッチインタフェースの設定]ダイアログでTS端子を割り当てます。 スマート・コンフィグレータでTS端子を使用する設定にしていない場合にはチェック機能が働きエラーとなります。
