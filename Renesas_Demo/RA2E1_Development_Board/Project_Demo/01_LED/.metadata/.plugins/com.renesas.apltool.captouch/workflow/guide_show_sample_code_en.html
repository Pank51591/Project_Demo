Implement the code to use capacitive touch.
Click on the "Show Sample" button to display the sample code and implement it in the main function of your project.
The sample code will change depending on the touch interface configuration, so if you change the touch interface configuration, be sure to change the implementationl.


File output :
The sample code can also be output as a file. In that case, call the "qe_touch_main" function in the main function.

Output location :

  • <project>\qe_gen

Output files :

  • qe_touch_sample.c


Notes on using gesture :
To use gesture, do the following actions.

  • Set to use the math library (math.h).
  • The heap size and stack size should be increased as needed.
  • When using the RA microcomputer, uncheck the following check box or increase the heap size, as the random function is used in AI preprocessing.
    GNU ARM Cross C Linker - [Use newlib-nano(--specs=nano.specs)]
