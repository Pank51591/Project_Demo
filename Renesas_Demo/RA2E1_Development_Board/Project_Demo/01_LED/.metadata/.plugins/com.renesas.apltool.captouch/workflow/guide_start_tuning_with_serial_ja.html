シリアル（UART）接続にて各タッチセンサの調整を自動で行います。
シリアル接続による調整を行うために、以下の手順を実施してください。

  1. スマート・コンフィグレータにてシリアル接続用の設定を行います。（ドライバの追加、端子の設定など。）
  2. 静電容量タッチミドルウェアのシリアル接続用の設定を行います。
  3. [調整用ファイルを出力する]ボタンを押し、調整処理に必要なファイルを出力します。
  4. main()関数内にタッチ用メイン関数を呼び出す処理を実装します。
  5. プロジェクトをビルドし、プログラムを実行します。
  6. ボーレート、ポート番号を指定し、[調整を開始する]ボタンを押してタッチセンサの調整を開始します。


調整の流れ：

  1. [調整を開始する]ボタンを押します。しばらくすると調整処理に入り、[自動調整処理中]ダイアログが表示されます。
  2. [自動調整処理中]ダイアログの指示に従い調整を行います。
  3. 例えばボタンの自動調整では、1つのボタンに対し1回の測定を行います。 ダイアログの指示に従い、対象のボタンに触れて、PCのキーボードのいずれかのキー(例えばYキー)を押してください。
  4. すべてのタッチインタフェースの測定が終ったら結果が表示されます。問題がなければ完了してください。 タッチインタフェースに触らずにPCのキーを押してしまった場合などは、再測定をするタッチインタフェースを選択後に[リトライ]ボタンを押し、再測定をしてください。


調整時の注意点：

  • [アドバンスドモード(高度な設定)]にチェックを入れると、調整に関する設定を行うことができます。ただし、誤った値を設定した場合や、明確な理解なしに設定した場合は、調整結果が悪くなる可能性があります。
