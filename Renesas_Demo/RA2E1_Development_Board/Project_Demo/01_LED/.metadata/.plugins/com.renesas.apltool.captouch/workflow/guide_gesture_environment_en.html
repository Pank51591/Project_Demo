Download the environment used for gesture development to your PC.
Click on the "Download" button in the "Status of Gesture Environment" dialog box that is displayed by clicking on the "Install Environment for Gesture" button.
Download the environment only if you are developing gestures for the first time, or if you are updating the environment.


Notes on environment installation:

  • To download the environment, a few GB of free space on your PC is required.
  • The environment includes Renesas AI conversion tool, e-AI translator (command line version), A license file is required to use the e-AI translator. Please obtain the file from the web page and place it in the specified folder. The file needs to be updated regularly.

    Web page :
    https://www.renesas.com/application/technologies/e-ai/tool

    Folder :
    <Support folder download area>\QE\AI_Package[index]\Tools\eAITranslator[Version]\bin

    You can check the location of the "support folder download area" by following the steps below.
    1. Click on the "Help" - "About e2 studio" menu in e2 studio.
    2. Click on the "Installation Details" button in the "About e2 studio" window.
    3. Open the "Support Folders" tab of the "e2 studio Installation Details" window and check the "e2 studio download area" path.
