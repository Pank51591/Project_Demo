Execute the user program with embedded touch middleware and confirm the sensitivity of the touch interfaces.
Clicking on the "Show Views" button switches to the monitoring perspective. By using the views for monitoring, you can check the sensitivity of the touch sensor and adjust parameters in real time.


How to monitoring :
For monitoring via serial (UART) connection, please follow the steps below.

  • Configure settings for serial connection in the Smart Configurator. (Add drivers, configure pins, etc...)
  • Configure the touch middleware for serial connection.
  • Run the program, specify the baud rate and port number, and click on the "Connect" button to make a serial connection.
  • After the serial connection is complete, click on the "Enable Monitoring" button in each monitoring view.
