【*】程序简介

-工程名称：FreeModbus-Slave例程
-实验平台: 野火STM32 F767 开发板 
-MDK版本：5.23
-ST固件库版本：1.24.0


【 ！】功能简介：
通过FreeModbus协议栈进行通信，可以用Modbus上位机连接开发板进行调试。
通过modbus_master_test.h中的宏定义选择具体功能。

学习目的：通过阅读Modbus官方文档结合历程学习FreeModbus通信原理。

【 ！！】注意事项：
考虑不同硬件平台原因，此工程默认使用串口通信，如果需要使用串口转485，只需要在usart.h中开启 MODBUS_MASTER_USE_CONTROL_PIN 宏定义即可。
 所以使用串口调试时，需要用用杜邦线连接USB转串口模块再接入电脑USB
 跳线帽：使用485通信时需要用跳线帽连接J40 (出厂默认已连接，在左上角)

/*********************************************************************************************/

【*】 引脚分配
    PD5     ------> USART2_TX
    PD6     ------> USART2_RX
	
/*********************************************************************************************/

【*】 时钟

A.晶振：
-外部高速晶振：25MHz
-RTC晶振：32.768KHz

B.各总线运行时钟：
-系统时钟 = SYCCLK = AHB1 = 180MHz
-APB2 = 90MHz 
-APB1 = 45MHz

C.浮点运算单元：
  不使用
